# Supabase Auth Template Notes

These HTML files are designed for Supabase hosted Auth email templates.

Important variables:

- `{{ .ConfirmationURL }}` is the secure link generated by Supabase.
- `{{ .Token }}` is the OTP code for OTP-style templates.
- `{{ .Email }}` is the recipient email.
- `{{ .SiteURL }}` is the configured site URL.
- `{{ .RedirectTo }}` is the redirect URL passed by the app.

Do not replace these variables with real values in the template.
